## 如何添加架构文档

运行 `flowforge docs-guide architecture` 获取写作指南。

每个架构文档应包含：系统分层、核心模块、技术选型、关键约束。
architecture/ 目录存放系统级架构分析，模块级设计放在 modules/<name>/ 下。
