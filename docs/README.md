# 项目文档

这里是 `FlowForge` 的长期文档主目录。

## 结构

- `explorations/`：提案创建前的探索工件
- `proposals/`：进行中和已归档的变更提案工作集
- `modules/`：按模块组织的最终文档
- `architecture/`：跨模块和系统级设计文档
- `decisions/`：稳定架构决策和 ADR
