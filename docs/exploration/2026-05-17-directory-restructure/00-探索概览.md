1. **源目录清晰**：不使用隐藏目录（`.claude/`），使用清晰目录名

3. **安装到项目**：其他项目安装时复制到隐藏目录
4. **自用模式**：tg-workflow 自身使用时通过软链接

## 探索结论

### 推荐目录结构

```
tg-workflow/
├── configs/                           # 清晰目录（非隐藏）
│   ├── claude/                        # Claude Code 平台配置
│   │   ├── commands/tg/               # /tg:* 命令
│   │   ├── skills/                    # 技能
│   │   ├── hooks/                     # 钩子
│   │   └── settings.json              # 配置
│   └── opencode/                      # OpenCode 平台配置
│       ├── commands/tg/
│       ├── plugins/
│       └── settings.json
├── docs/
├── templates/
├── scripts/install.sh
└── AGENTS.md
```

### 安装模式

| 模式 | 命令 | 说明 |
|------|------|------|
| 项目安装 | `./install.sh project [path]` | 复制到项目的隐藏目录 |
| 全局安装 | `./install.sh global` | 复制到用户目录 |
| 自用模式 | `./install.sh self` | 创建软链接 |
