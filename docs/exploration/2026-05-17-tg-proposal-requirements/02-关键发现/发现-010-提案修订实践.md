**问题**：

### 方向 3：Beads 任务联动

**方案 A：创建修订任务**
```bash
bd create "修订：修正命令实现方式" \
  --parent {epic-id} \
  --spec-id "CR26051701" \
  --metadata '{"type": "revision", "version": "v1.1"}'
```

**方案 B：重新打开已完成的任务**
```bash
bd reopen {task-id} --reason "实施发现缺陷，需要修正"
```

**方案 C：创建修订 Epic（独立于原提案）**
```bash
bd create "CR26051701-r1: 修订命令实现" \
  --type epic \
  --spec-id "CR26051701-r1" \
  --metadata '{"parent_proposal": "CR26051701"}'
```
