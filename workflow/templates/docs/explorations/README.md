# Exploration Template

Canonical structure for a new exploration:

```text
docs/explorations/<slug>/
├── index.md
├── journal/
│   └── 2026-05-20-entry.md
├── findings/
│   └── F-001-example.md
├── decisions/
│   └── D-001-candidate.md
└── artifacts/
```

Use `index.md` as the main reading surface.
