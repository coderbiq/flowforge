# 探索笔记目录

本目录用于存储早期探索阶段的笔记，记录想法、调研和讨论过程。

## 目录说明

探索笔记是**低正式度**的文档，用于：
- 记录初步想法和假设
- 整理调研结果
- 讨论问题和可能的解决方案
- 为正式提案做准备

## 命名规范

```
YYYY-MM-DD-topic-name.md
```

示例：
- `2026-05-17-database-selection.md`
- `2026-05-18-auth-strategy.md`

## 状态定义

| 状态 | 说明 |
|------|------|
| 探索中 | 正在调研和讨论 |
| 已定稿 | 探索完成，可以创建正式提案 |
| 已废弃 | 不再继续探索 |

## 工作流

```
探索笔记 → 想法成熟 → 创建提案 (/tg:propose)
          → 不再继续 → 标记废弃
```

## 与其他文档的关系

- 探索笔记 → **提案** (`../proposals/`)
- 重要的架构决策 → **ADR** (`../decisions/`)
- 关键发现 → **长期记忆**（自动存储）
